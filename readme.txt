== Description ==

Creates a shortcode for dropping in a Google Calendar. The plugin uses your WordPress Timezone by default. Example: [pydgooglecal account="YOUR_CAL@ACCOUNT.COM" width="800" height="600"].

== Installation ==

1. Place the 'pyd-cal_shortcode' folder in your '/wp-content/plugins/' directory.
2. Activate Google Calendar Shortcode.
3. Use Shortcode [pydgooglecal account="YOUR_CAL@ACCOUNT.COM" width="800" height="600"] on any page or post you want to display the calendar
